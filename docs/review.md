## Description

SpacePeek combines a high-performance Rust scanning engine with a polished SwiftUI interface to deliver ultra-fast folder analysis on macOS. Select a folder in Finder, press Space, and instantly inspect folder and file counts, total size, and top space-consuming items.

Built for quick exploration, SpacePeek lets you navigate through subfolders and understand storage distribution at multiple levels without delay.
