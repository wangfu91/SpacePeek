## Description

Understand your folders at a glance.

SpacePeek brings fast, visual folder analysis directly to Finder Quick Look. Select any folder and press Space to instantly see its size, file and folder counts, largest contents, and storage distribution—without opening another app.

Built for developers and power users, SpacePeek makes it easy to understand what’s taking up space and navigate large directory trees with speed and clarity.

Features

• Instant folder previews from Finder Quick Look
• Ultra-fast scanning engine built for large folders
• Total size, file count, and folder count at a glance
• Largest files and subfolders ranked by size
• Interactive storage distribution charts
• File age and content type analysis for Downloads folders
• Automatic Git repository detection and metadata
• Copy folder paths with a single click
• Fully local processing. No indexing, no uploads, no tracking.

SpacePeek Pro

Unlock unlimited drill-down navigation and explore subfolders to any depth directly inside the preview window.

Whether you’re inspecting project directories, investigating storage usage, or simply curious about what’s inside a folder, SpacePeek helps you understand your files instantly.
