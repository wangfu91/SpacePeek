# SpacePeek

SpacePeek is a macOS Quick Look extension for folders that helps you inspect folder size and storage usage instantly from Finder.

## What it does

- Preview total size and item counts for any selected folder
- Inspect the biggest files and subfolders using a local, fast scan
- Open the preview directly from Finder with the Space Bar
- Keep scanning and preview work on your Mac with no uploads or cloud processing

## Highlights

- Built for macOS 14.0 or later
- Uses a high-performance Rust scanning engine and polished SwiftUI interface
- Designed for quick storage insight without opening a heavy disk utility
